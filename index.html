HTML - Introdução


O que é HTML?


HTML, sigla para HyperText Markup Language, é a linguagem padrão usada para criar páginas da web e aplicativos da web. É uma linguagem de marcação que define a estrutura básica e o conteúdo de uma página da web, permitindo que os navegadores da web interpretem e exibam o conteúdo de forma adequada para os usuários.

Imagine que você está construindo uma casa de brinquedo com peças de LEGO. O HTML é como o plano ou as instruções que você segue para colocar as peças de LEGO juntas da maneira certa.


Tags HTML


As tags HTML são elementos fundamentais na linguagem HTML que permitem estruturar e formatar o conteúdo de uma página da web. Elas são usadas para marcar diferentes partes do conteúdo, fornecendo instruções ao navegador de como ele deve exibir esses elementos na página. Uma tag HTML é uma sequência de caracteres envolvida por colchetes ("<" tag ">" ). As tags podem conter um nome, que identifica o tipo de elemento, e podem incluir atributos, que fornecem informações adicionais sobre esse elemento. Imagine que você tem vários tipos de peças de lego coloridas. Cada tipo de peça tem uma forma e cor diferentes. As tags HTML são como esses peças de lego. Cada tipo de peça (tag) tem um propósito especial. Por exemplo, temos peças para fazer partes de texto, imagens e links.


Como é formada uma Tag HTML?


Um elemento HTML é definido por uma tag inicial, um conteúdo e uma tag final:

<< /span>tag>Conteudo << /span>/tag>

Exemplo

<!DOCTYPE html>
<html>
<head>
<title>Nome da Página </title>
</head>
<body>
<h1>Título da Página</h1>
<p>Parágrafo</p>
</body>
</html>

Principais Tags HTML

  • <!DOCTYPE html>Define a versão do HTML que está sendo usada
  • <html>Define o elemtno raiz de um documento HTML</html>
  • <head>Contém as informações sobre o documento, como meta informações e links para arquivos externos, como por exemplo CSS e Javascript</head>
  • <title>Define o título da página que aparece na barra de título do navegador</title>
  • <p>Cria parágrafos de texto</p>
  • <a>Define hiperlinks para outras páginas</a>
  • <ul>Cria uma lista não ordenada</ul>
  • <ol>Cria uma lista ordenada</ol>
  • <li>Define um item de lista em listas ordenadas ou não ordenadas</li>
  • <table>Cria uma tabela</table>
  • <tr>Define uma linha em uma tabela</tr>
  • <td>Define uma célula de dados em uma tabela</td>
  • <th>Define uma cabeçalho de célula em uma tabela</th>
  • <form>Cria um formulário para entrada de dados</form>
  • <input>Define um campo de entrada em um formulário</input>
  • <label>Define uma etiqueta para um campo de formulário</label>
  • <img>Insere uma imagem

CSS


CSS, ou Cascading Style Sheets, é uma linguagem de marcação utilizada para definir o estilo e a apresentação de páginas da web. Com o CSS, é possível controlar a aparência de elementos HTML, como cores, fontes, layouts e espaçamentos, permitindo assim criar interfaces visuais atraentes e bem organizadas para os usuários.


Sintaxe CSS


A sintaxe do CSS segue um padrão específico que envolve a definição de seletores, propriedades e valores.


Seletores:

Os seletores são padrões que definem quais elementos HTML serão estilizados pelo CSS. Eles podem ser elementos HTML, classes ou IDs. Os seletores são seguidos de chaves {} que contêm as propriedades e valores.


Propriedades:

As propriedades são características que você deseja estilizar, como cor, fonte, tamanho, margem, etc.


Exemplo

                            h1{
                                color: #a21caf;
                            }
                            p{
                                color: #65a30d;
                            }

Tipos de seletores


  • Elemento HTML: O seletor de elemento aplica estilos a todos os elementos correspondentes. É simplesmente o nome do elemento HTML que você deseja estilizar. Como no exemplo anterior.
  • Classe: O seletor de classe é precedido por um ponto (.) e é usado para estilizar elementos que têm uma determinada classe atribuída. 
                                    .titulo{
                                    color: #a21caf;
                                }
                                .paragrafo{
                                    color: #65a30d;
                                }
  • ID: O seletor de ID é precedido por um hash (#) e é usado para estilizar um elemento HTML específico com um ID único. 
                                    #cabecalho{
                                    font-size: 24px;
                                }

Como inserir o CSS no meu projeto?


Há várias maneiras de inserir CSS em um documento HTML para estilizar suas páginas da web.


CSS Embutido(Inline)

Você pode incluir estilos diretamente em um elemento HTML usando o atributo style. Os estilos serão aplicados apenas a esse elemento específico.

<h1 style="color: red;">Título da Página</h1>

CSS Incorporado (Embedded CSS):


É possível adicionar estilos dentro da tag head do seu HTML. Esses estilos se aplicarão a todo o documento.


<head>
<style> 
                        
                            body{
                                background-color: green;
                            }
</style>
</head>

CSS Externo


Você também pode criar um arquivo CSS separado contendo todas as regras de estilo e, em seguida, vinculá-lo ao seu HTML usando a tag "link".

<head>
<link rel="stylesheet" href="style.css">
</head>
